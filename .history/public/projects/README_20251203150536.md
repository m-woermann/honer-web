# Project Images Needed

To complete the projects system, you'll need to add these image files to `/public/projects/`:

## The Atrium Project
- `atrium-main.jpg` - Main project hero image (for banner and card)
- `atrium-detail.jpg` - Detail/close-up shot 
- `atrium-overview.jpg` - Wide overview shot

## Studio Complex Project  
- `studio-main.jpg` - Main project hero image (for banner and card)
- `studio-detail.jpg` - Detail/close-up shot
- `studio-mixing.jpg` - Mixing room shot

## Image Requirements
- **Resolution**: 1920x1080 or higher recommended
- **Format**: JPG or PNG
- **Aspect Ratio**: 16:9 works best for the banner expansion effect
- **File Size**: Optimize for web (under 2MB each)

## Adding New Projects
1. Create new JSON file in `/src/config/projects/[project-name].json`
2. Add project to `/src/config/projects.json` 
3. Update the project configs object in both `portfolio.astro` and `project-detail.astro`
4. Add corresponding images to `/public/projects/`
